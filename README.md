# contact-leyu-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源的形式存放，不绑定任何特定域名或网站。

## 项目简介

这是一个通用型 HTML 页面归档仓库。仓库内容主要为独立的、可离线查看的 HTML 文件，适用于需要保存或分发静态页面的场景。所有页面均为自包含资源，不依赖外部服务或动态后端。

## 目录结构说明

```
contact-leyu-resource/
├── pages/          # 存放主要 HTML 页面文件
├── assets/         # 页面引用的静态资源（如样式表、图片、脚本等）
├── docs/           # 相关文档或说明文件（可选）
└── README.md       # 本文件
```

- **pages/**: 每个 HTML 文件为一个独立页面，文件名应尽量反映页面内容。
- **assets/**: 按类型或页面组织，保持资源与页面之间的引用关系清晰。
- **docs/**: 存放与页面内容相关的辅助文档或使用说明。

## 页面归档说明

- 所有页面均为静态 HTML，不含后端逻辑。
- 页面之间无直接依赖关系，可单独复制使用。
- 归档遵循“一次提交，长期保存”的原则，不频繁修改已归档页面。
- 新页面添加时，建议附带简要说明（可在 commit message 或 docs 目录中注明）。

## 维护说明

- 本仓库由维护者根据需求不定期更新。
- 欢迎提交 Issue 或 Pull Request，但请注意：不接受涉及具体网站、商业推广或违规内容的页面。
- 提交新页面时，请确保 HTML 文件结构完整、无外部不可达资源。
- 如需引用仓库内资源，请使用相对路径。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/contact-leyu-resource.git
   ```
2. 直接打开 `pages/` 目录下的 HTML 文件即可查看。
3. 也可部署到任何静态文件服务器上使用。
